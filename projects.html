Spending Insight Pro Model üíµüìäüîçüìà

Monthly expenditure analysis using Data Science.
This project uses numpy, pandas, seaborn, and matplotlib libraries to build a model that analyzes spending categories, amounts, balances, and purchase dates. The data, collected from bank statements, is visualized using PowerBI.

Steps performed to implement the model:
1. Collected data on expenditures, such as amount spent, category, and date.
2. Uploaded the data into a Python environment using the Pandas library.
3. Performed exploratory data analysis to understand the data and its distribution.
4. Visualized the data to identify patterns and relationships.
5. Performed feature engineering to extract relevant information.
6. Pre-processed the data by dropping irrelevant columns and encoding categorical variables.
7. Split the data into training and testing sets.
8. Built and trained a regression model using the Random Forest algorithm.
9. Evaluated model performance using metrics such as Root Mean Squared Error.
10. Used the model to predict expenditure amounts for new data points.
