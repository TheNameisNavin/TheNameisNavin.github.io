Spending Insight Pro Model

Monthly expenditure analysis using Data science. This project involves using numpy, pandas ,seaborn and matplotlib libraries in order to build a model that analyzes the category, amount spent, balance amount and the date on which the purchase was made. The data set was collected from a bank statement of each month and then uploaded.The report is visualized using PowerBI.

Steps performed to implement the model: 1. Collected data on expenditures, such as amount spent, category, and date.
2. Uploaded the data into a Python environment using the Pandas library.
3. Performed exploratory data analysis to understand the data and its distribution.
4. Visualized the data to identify patterns and relationships.
5. Performed feature engineering to extract relevant information from the data.
6. Pre-processed the data by dropping irrelevant columns and one-hot encoding categorical variables.
7. Split the data into training and testing sets for model building.
8. Built and trained a regression model using the Random Forest algorithm.
9. Evaluated the model performance using a metric such as Root Mean Squared Error.
10.Used the trained model to predict expenditure amounts for new data points, such as for a specific category and date.
